```python
class VLLMConfig(BaseModel):
    """vLLM backend configuration.

    Nested structure mirrors vLLM's own two-API separation:
    - engine: vllm.LLM() constructor arguments (engine-level, loaded at model init)
    - sampling: vllm.SamplingParams arguments (vLLM-specific extensions only)

    Universal sampling params (temperature, top_p, top_k, repetition_penalty)
    live in DecoderConfig and are shared across all backends.

    Example YAML:
        backend: vllm
        vllm:
          engine:
            enforce_eager: false
            gpu_memory_utilization: 0.9
            kv_cache_dtype: fp8
          sampling:
            max_tokens: 512
            presence_penalty: 0.0
    """
    model_config = {"extra": "forbid"}
    engine: VLLMEngineConfig | None = Field(default=None, description="Engine-level configuration: vllm.LLM() constructor args")
    sampling: VLLMSamplingConfig | None = Field(default=None, description="Sampling-level configuration: vllm.SamplingParams extensions (vLLM-specific only)")
```

Current broken code (line 347):
```python
config_dict.setdefault(backend, {})[dim_key] = value
```

Replace with:
```python
from llenergymeasure.config.loader import _unflatten, deep_merge
# ...inside the for dim_key, value loop:
if dim_key in backend_scoped:
    # Backend-scoped parameter: recursively unflatten to handle multi-level paths
    # "engine.block_size" -> {"engine": {"block_size": value}}
    backend_dict = config_dict.get(backend, {})
    nested_update = _unflatten({dim_key: value})
    config_dict[backend] = deep_merge(backend_dict, nested_update)
```
