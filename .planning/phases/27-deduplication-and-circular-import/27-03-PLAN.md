<!-- Correct nvml_context() pattern (from RESEARCH.md pitfall #4) -->
```python
from contextlib import contextmanager
from typing import Generator

@contextmanager
def nvml_context() -> Generator[None, None, None]:
    """Context manager for NVML init/shutdown lifecycle.

    Best-effort: silently ignores ImportError (pynvml not installed) and
    NVMLError (no NVIDIA GPU). Callers receive None on failure - handle gracefully.
    """
    try:
        import pynvml
        pynvml.nvmlInit()
        try:
            yield
        finally:
            pynvml.nvmlShutdown()
    except Exception:
        yield  # pynvml absent or nvmlInit failed — caller proceeds without NVML
```
IMPORTANT: The inner try/finally ensures shutdown is called after yield. The outer try/except catches only nvmlInit errors. This preserves exceptions that occur inside the `with` block.
