<objective>
Fix two integration issues found by the v0.9.0 milestone audit:

1. **Preflight runner resolution** — `run_study_preflight()` calls `resolve_study_runners(list(backends))` without passing `yaml_runners` or `user_config`, so preflight runner resolution can diverge from the actual resolution in `_api._run()`.

2. **GPU memory check in single-experiment path** — `_run_in_process()` does not call `check_gpu_memory_residual()` before running an experiment, unlike the multi-experiment `StudyRunner` path.

Purpose: Close two audit-flagged integration gaps so that preflight checks and GPU memory monitoring behave consistently regardless of how experiments are dispatched.
Output: Patched source files + unit tests confirming the fixes.
</objective>
